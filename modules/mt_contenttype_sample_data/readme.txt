# Content Type Sample Data Module
To test this module:
- install the module

# Test Import
- go to Configuration -> MT Content Type Sample Data
- choose the file for import from module core directory (mt_simple_data_3_items.json)
- click the Import button
- go to the Content
- You will see a nodes with MT Product content type

# Test Export
- create a few MT Product nodes
- move nodes pictures to module "img" folder
- go to Configuration -> MT Content Type Sample Data
- enter the file name in the field
- click the Export button
- go to folder -> site_name/sites/default/files/MTSampleData
- You will see the imported file_name.json file with json data
